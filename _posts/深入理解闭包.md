什么是闭包？
> 函数在执行的时候就会形成一个私有作用域，而这个私有作用域内的变量是不受外界干扰的。我们把这种私有作用域保护私有变量不受外界干扰的机制就叫闭包。

# 如何制造闭包?
有一个执行的函数
```
//制造闭包，保护变量n不受到外界干扰：
(function (){
    var n = 0;
    document.body.onclick = function (){
        n++;
        document.getElementsByTagName('title')[0].innerHTML = n;
    }
})();
```
